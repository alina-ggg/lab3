# Лабораторная работа №3: Программирование циклического вычислительного процесса

Данный проект выполнен в рамках учебной дисциплины «Программирование». В работе реализованы алгоритмы циклической структуры для решения математических и прикладных задач на языках **C++** и **Go**.

## 🎯 Цели работы
Изучение циклических алгоритмов, операторов цикла (`for`, `while`, `do-while`) и программирование сложных итерационных процессов.

## 🛠 Реализованные задания
В соответствии с вариантом выполнены следующие задачи:

1. **Табулирование функции:** Вычисление и вывод в виде таблицы значений функции, заданной графически, на интервале от $X_{нач}$ до $X_{кон}$ с шагом $dx$.
2. **Тест на простоту:** Реализация алгоритмической проверки натуральных чисел на простоту (Тест Поклингтона).
3. **Математическая сумма:** Расчет суммы бесконечного ряда с контролем точности вычислений.
4. **Увлекательная игра:** Моделирование игровой стратегии (задача о Павле и Вике) с использованием циклов.
5. **Моделирование остывания кофе:** Решение задачи об остывании чашки кофе из кофейни НГТУ. Программа моделирует процесс по закону Ньютона и строит таблицу зависимости температуры от времени.

## 💻 Стек технологий
* **C++**: Работа с манипуляторами вывода `iomanip` для создания аккуратных таблиц, использование математической библиотеки `cmath`.
* **Go**: Реализация аналогичной бизнес-логики, демонстрация работы со срезами (slices) и форматированным выводом `fmt`.
